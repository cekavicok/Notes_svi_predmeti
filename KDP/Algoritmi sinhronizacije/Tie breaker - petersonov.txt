ZA N PROCESA

mogu da prodjem ako sam jedini ili ako nisam prolednji

in[i] - koje stanje proces CS[i] prolazi
last[j] - koji proces je poslednji dosao u datu iteraciju

int in[1:n] = ([n] 0);
int last[1:n] = ([n] 0);
process CS[i=1 to n] { //i je broj procesa, j je broj iteracije tj "nivo"
    while(true){
        for(j=1 to n){
            in[i] = j;
            last[j] = i;
            for(k=1 to n; i != k) {
                while(in[k]>=in[i] and last[j]==i) skip; 
                //dok god je k-ti proces na >= nivou od i, i proces i je poslednji na nivou j
            }
        }
        KS;
        in[i] = 0
        not KS;
    }
}

Poenta algoritma, svaki proces prolazi n nivoa (j su nivoi, a i su procesi)
k su ostali procesi, i se nece pomerati dalje dok postoji drugi proces koji je na istom 
ili vecem nivou ili vise nije poslednji
