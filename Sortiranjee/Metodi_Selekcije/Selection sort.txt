Selection_sort(array, n){
    for(int i=0; i<n-1; i++){
        int min = array[i];
        pos = i;
        for(int j = i+1; j<n; j++){
            if(min > array[j]){
                min = array[j];
                pos = j
            }
        }
        if(pos != i){
            array[pos] = array[i];
            array[i] = min;
        }
    }
}

Performanse: O(n^2) - O(n^2)
Mem: O(1)
